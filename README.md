## ğŸŒŸ Overview

**RENIS Store** is a modern e-commerce platform designed for premium men's innerwear. 

This repository hosts the **Static Frontend** version of the application. It is a high-performance, single-page application (SPA) built with **React** and **TypeScript**, utilizing **Mock Data** to demonstrate the user interface and shopping experience without requiring a live backend server.

## ğŸš€ Getting Started

Follow these steps to run the project locally.

### Prerequisites

- **Node.js** (v18 or higher)
- **npm** (v8 or higher)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/madhesh04/Renis-Store.git
   cd Renis-Store
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

### Running the Application

Start the development server:

```bash
npm start
```

- Open your browser and navigate to: `http://localhost:5173`
